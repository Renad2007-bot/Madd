# COM — منصة "مدّ" الطلابية 🎓

منصة طلابية متكاملة تجمع بين واجهة ويب تفاعلية وخلفية برمجية (Back-end) مدعومة بالذكاء الاصطناعي لتقديم ثلاثة خدمات رئيسية:
* **استخراج النصوص (OCR):** قراءة النصوص من صور كشوف الدرجات.
* **تصنيف النصوص (Classification):** الكشف عن النصوص المسيئة باللغة العربية باستخدام نموذج **AraBERT**.
* **التنبؤ بالتخصص (Prediction):** ترشيح أفضل 3 تخصصات جامعية تناسب الطالب باستخدام نموذج **RandomForest**.
